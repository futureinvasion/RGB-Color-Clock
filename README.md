RGB Color Clock
---

iOS application written in [Swift](https://developer.apple.com/swift/) display background color (Hex Code) of digital clock based on current time
	
	view.backgroundColor = UIColor(rgba: #150135)
	
when the time is
	
	15 : 01 : 35
